About The Event

The PyConNG 2019 is the largest annual gathering of Python technology users including developers, data scientists/analysts, business intelligence experts, representative of organizations and everyone interested in technology. It is organised by members of the Python Users Nigeria Group, a nonprofit organization dedicated to advancing the use of Python technology in Nigeria through her community initiatives.

Participants come together to network, collaborate and learn via the various high impact sessions. PyconNG 2019 will feature tutorial sessions, hackathon, sprint, keynote addresses, and various talk sessions in data science, Web/Devops, and general talks.

The PyConNG 2019 will mark the 3rd edition of the annual pycon conference with the last two conferences which was held in Lagos, hosting over 300 participants from Africa, America and different states in Nigeria. The 2019 Pycon Nigeria conference promises to be bigger and better.

The PyConNG 2019 conference will also provide a platform for companies/organizations to exhibit their product and service offerings. We look forward to host you in Nigeria’s federal capital territory for the 2019 PyconNG conference!

Event Features

Here are the exiting event features and sessions

Keynotes

The keynote session will feature keynote addresses on topical issues around the use of python technology as well building a scalable business/startup from coding.

Tutorials

The tutorial sessions are design to help aspiring python developers, aspiring data scientists and technology enthusiasts develop their skills and expertise in Python programming.

Hackathon

The goal of the hackathon is to leverage the simplicity and efficiency of the python programming language to solve a Nigerian-centric problem.

Specialized talks

This session will feature various talks on using Python for exploratory & Data Science, Building web apps with Python frameworks, CI/CD (DevOps)
